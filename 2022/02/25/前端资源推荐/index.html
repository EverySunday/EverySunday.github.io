前端资源推荐

1. 编辑

​ http://runjs.app/ 实时输出结果的 app

​ http://jsrun.net 需要注册的在线编辑器

​ http://enjoycss.com/ 一个完整的代码生成器，可以自定义输入域或CSS3按钮

​ http://codepen.io/ 网站里有很多很酷的特效，而且可以看到效果的源代码，也可以看到实现效果

​ https://neumorphism.io/#e0e0e0 在线UI编辑

​ https://uiverse.io/ 好看的UI元素，开源免费

2. HTML5

​ http://www.html5tricks.com/ 网站里有很多前端实现的功能，可以在线查看效果，也可以下载源码的

​ http://caniuse.com/ 查询浏览器兼容性的网站

​ https://www.minimamente.com/ 集成各种特殊动效的动画库

3. JS

​ http://jshint.com/ 一个在线JS检测工具，可以检测JavaScript代码中的错误和潜在问题

​ https://easings.co/ 计算交互作用，例如：轮播图、侧面菜单、滚动、底部彩灯、横态等

​ https://animista.net/ 庞大的动画库。你可以找到可用于组件，照片和文本的基本，以及更高级的动画

​ https://tympanus.net/Development/MagneticButtons/index.html 有趣的悬停动画的磁性按钮

​ https://svgwave.in/ 波浪效果生成器。可以制作多个图层并对其进行修改

​ https://greensock.com/gsap 动画

​ https://underscorejs.org 使用一系列不同的函数来执行 JavaScript 中的各种编程任务

​ https://www.chartjs.org 统计

​ https://d3js.org 在线数据操作文档，数据可视化

​ https://lodash.com 轻松处理数字、字符串、对象和数组

​ https://github.com/alvarotrigo/fullPage.js 帮助你轻松创建具有一页的全屏滚动网站

4. CSS
（1）按钮

​ http://www.bestcssbuttongenerator.com/ 这个网站主要提供各种按钮的CSS代码

（2）图标

​ https://www.iconfont.cn/ 阿里巴巴矢量图标库

​ https://www.iconshock.com/ 渐变图标

（3）背景

​ https://bansal.io/pattern-css 使用CSS库完成图案填充背景效果

​ https://www.magicpattern.design/tools/css-backgrounds 纯CSS背景图案

（4）鼠标悬停

​ http://ianlunn.github.io/Hover/ 鼠标悬停动画

（5）布局

​ https://grid.layoutit.com/ 在线生成网格

​ https://csslayout.io/ 102种可选择模式

​ https://loading.io/ Flexbox布局

​ https://cssgr.id/ 指定行数和列数生成代码

（6）综合

​ https://css-tricks.com/ 一个关于运用CSS技术进行网络开发和设计的小技巧的交流网站，里面有各类样式问题的解决方案以及深度思考

​ https://animate.style/ 简单便捷CSS动画库

​ http://elrumordelaluz.github.io/csshake/ 震动和晃动DOM元素的CSS库

​ https://kushagra.dev/lab/hint/ 一个用纯CSS和HTML实线的提示库

​ https://hiloki.github.io/kitecss/ 一个兼容性极好且灵活的布局CSS库

​ https://www.shapedivider.app/ 创建可响应的波形和自定义形状分割线

​ https://bennettfeely.com/ clip-path属性允许你通过将元素裁剪为各种基本形状（圆形，多边形或插图等）

​ https://cssgradient.io/ 渐变代码生成器
