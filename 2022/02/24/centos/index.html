在虚拟机Vbox中安装CentOS,并安装GCC
（1）进入centos命令端，输入：
vi /etc/sysconfig/network-scripts/ifcfg-eth0

并修改为以下内容，保存退出后，重启（reboot）：

（2）执行以下命令，安装gcc、c++编译器及内核文件:
yum -y install gcc gcc-c++ kernel-devel
（3）测试是否安装成功：
vi test.c

写入以下内容，并保存退出：

#include <stdio.h>

int main()
{
        printf("I love FishC.com!\n");
        return 0;
}

输出以下命令看是否成功打印：

gcc test.c -o test && ./test
（4）安装图形界面：

​ 安装X窗口：

yum -y groupinstall "X Window System"

​ 安装GNOME：

yum -y groupinstall "Desktop"
（5）设置GNOME为默认启动：
vi /etc/inittab

​ 将下图光标位置的3改成5，保存并退出：

（6）创建一个用户账户（为了安全），并重启：
useradd -d /home/sunday -m sundaypasswd sunday
